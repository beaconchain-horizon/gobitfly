# Security Policy

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability within any gobitfly project, please **do not** report it through public GitHub issues.

Instead, please send an email to:

- `beaconchain@beaconchain.us`
- `gamma.mahdii@gmail.com`

You should receive a response within 48 hours. If the issue is confirmed, we will release a patch as soon as possible depending on complexity.

## Supported Versions

Only the latest stable version of each project receives security updates. You are encouraged to always run the latest release.
